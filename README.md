# 学生就业管理系统 python1230

学生就业管理系统是基于Python 3.9和Django框架，配合MySQL 8数据库开发的一套源代码开放的管理系统。它提供了丰富的功能模块，旨在帮助学校和企业高效管理学生的就业信息。

## 技术栈

- Python 3.9
- Django
- MySQL 8

## 功能模块

- **学院信息管理**：管理各学院的详细信息。
- **专业信息管理**：对各个专业的数据进行维护。
- **企业信息管理**：录入和管理企业合作信息。
- **岗位信息管理**：提供岗位信息的增加、删除和修改功能。
- **学生信息管理**：维护学生基本信息。
- **学生教育经历**：记录和跟踪学生的学习经历。
- **学生项目经历**：管理和展示学生的项目实践经验。
- **学生投递记录**：跟踪学生的岗位申请记录。
